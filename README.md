## Getting Started

   To get a local copy up and running follow these simple example steps.

### Prerequisites

    NodeJS
### Setup

    1. git clone https://github.com/ntzwilly/To_Do_List.git
    2. cd To_Do_List

### Install

    npm install

### Usage

    npm start
